The Folder Encrypter is a Python-based application designed to provide secure folder encryption and decryption using the Fernet symmetric encryption method. This application features a user-friendly graphical interface built with Tkinter, enabling users to easily encrypt or decrypt all files within a selected folder. The encryption key is securely emailed to a specified recipient using the smtplib library, ensuring the key's safe distribution. Additionally, the application includes functionality to open a details web page, enhancing user experience. This project integrates several key Python libraries, including cryptography, tkinter, os, and webbrowser, making it a comprehensive solution for folder security and key management.
